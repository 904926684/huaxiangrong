# huaxiangrong

目标：花想容个人汉服博客展示，Hanfu personal portrait website shows.

作用：Blog就是以网络作为载体，简易迅速便捷地发布自己的心得，及时有效轻松
地与他人进行交流，再集丰富多彩的个性化展示于一体的综合性平台，此平台是展
现汉服女子写真，流露中国古韵的美。

功能：展示自己写真平台的自己交互式博客。
项目安装：此博客是前端页面，没有做后台数据库连接，无需繁杂的安装，直接打
开可行。

使用指南：这是一个单页面平台，分为三个模块，可根据所点击的模块进行滑动跳
转，第二个模块可通过编写时候预设筛选展示图片，可查看原图，鼠标滑过字体可
简单响应。
